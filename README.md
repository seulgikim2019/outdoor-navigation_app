# outdoor-navigation_app
실외지도(네비게이션 포함) 어플


기능
----
1. 구글지도 검색
2. 소요시간, 경로 확인
3. 네비게이션

사용기술
--------
● 언어: Java 

● API: Google MAP, Mapbox API
