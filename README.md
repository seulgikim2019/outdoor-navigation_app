# outdoor-navigation_app
 일본 총무성이 주최한 '다국어 번역 대회' 본선 진출작으로, 
 
 기본적인 통역ㆍ번역 서비스와 사용자 상황(날씨,위치 기반)을 고려한 AI 챗봇, 위치 서비스를 제공하는 여행 어플입니다.
 
 저는 해당 프로젝트에서 위치와 관련된 기능을 담당하였습니다. 실외에서 GPS의 정확도를 높이기 위해 Fused API를 사용하여 개발하였습니다. 
 
 여행 중 지도와 관련된 별도의 어플을 사용하지 않을 수 있도록 위치 및 경로 검색(Google MAP API)과
 
 네비게이션 기능(Mapbox API 사용)이 제공되는 실외지도를 개발하였습니다.



기능
----
1. 구글지도 검색
2. 소요시간, 경로 확인
3. 네비게이션


사용기술
--------
● 언어: Java 

● API: Google MAP, Mapbox API
